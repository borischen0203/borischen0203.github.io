英國求職 - Junior software engineer的求職紀錄 (3) BBC敗部復活

By Boris 2022-06-28 英國求職

3 月底 Onboard Accolite Digital 後，我以為找到工作後就可以不用一直面試，結果在派遣公司工作還要跟客戶面試，客戶覺得你能力 ok 才能加入他們 team。同一個客戶不同的 team 還要另外面試。覺得這真的是非常蠢的一件事…

我預計被派去的客戶端是 BT(British Telecom)，跟兩個不同的 team 面試後，被第二個 team 選上，但是可能一個月要進一次 office，而且這個 office 在東英格蘭，來回一趟要 6 小時，WTF?? 這跟當初面試說好的 full remote 不一樣啊…??

種種原因，更讓我堅信不會在這待太久，也好在沒有被 hunter 誆騙推掉其他面試。

之後陸續面了 2 間，BBC 和 Starling bank

BBC 面試共 2 關:

• 第一關 — Take-home assessment
做一個 command line tool，讓 user 可以 input 複數的網址， 並用 Json 格式列出對這些網址發送 HTTP GET request 時回傳的 Status code 和 Content-length。

成品: 【 https://github.com/borischen0203/Get2Json 】

• 第二關 — Technical interview + Culture fit
面試官有兩位，Team leader 和 一位 software engineer，Technical 的部分主要是 pair-programming，面試官請我先解釋一下我的 code 的邏輯和運作，因為他們不會 Golang(要求是用 Java 或 Python，但我寫信去問能不能用 Golang，他們也說 ok！)。

面試官對我做的一個小東西相當有興趣，問我這是什麼東西? 因為我在程式裡預設了當你什麼東西都沒輸入時會印出一個 gopher 的圖案

「Well, you found the easter egg！」我說。

面試官們整個大笑，似乎非常喜歡這個彩蛋也加深了他們對我的印象。

解釋完邏輯和流程後，便開始 pair-programming 請我優化一些功能和 test case，接著問我「 Java 和 Golang 之間有什麼差異? 」，culture fit 的部分則是問我「 為什麼想投遞這個職缺和這個 team」和「 你認為怎樣才稱得上 excellent engineer?」，整個面試大概 2 個小時多。

大概過了一週後就收到來自 Team leader 的 reject letter，不過信上有提到我是 45 個申請人裡排 Top 3，問我可不可以把 CV 分享給其他的 hiring managers。

看到 Top 3 這種東西，覺得大概是罐頭信。沒想到一個月後又收到主管來信其他 team 有人離職，因此現在有釋出職缺問我還有沒有興趣？

回覆信件後，主管便說會再和團隊討論，因為我之前已經面過相似的職缺了，這次面試將會非常輕量化。

過一週就突然就收到 offer 啦，連面試都不用面了！主管表示討論後不用在面試了，上次開缺只有 2 個名額，所以我被 Reject，但申請者的資料都會在系統留存 4 個月，所以這次找上第三順位的我。

才剛在派遣公司 onboard 2 個月就因為這突如其來的意外跳到 BBC 了。

後記: onboard 後才知道，主管當初還有寄信給幾位申請人，但只有我回信，我就在沒對手的情況下直接晉級 😂
