英國求職 - Junior software engineer 的求職紀錄 (2) 第一個真正的 offer

By Boris 2022-06-20 英國求職

被 Lloyds Bank 用很爛的理由 reject 後，雖然很崩潰，但我實在沒有時間灰心喪志，簽證效期只剩一年半，又要找 Junior 職缺，這兩個因素勢必會提高找求職的難度，或許是因為運氣好，這次投了 33 間面 9 了間，花了約 3 週多就收到 offer。

剛好有一間印度派遣公司(Accolite Digital)在英國開了新分部(成立約一年多)，要收可以立馬上工的 Graduate software engineer，對他們來說我半年的年資也算是 Graduate 的範圍裡。一般 Graduate software engineer 都是給應屆畢業或是正在就讀的學生，在 3 月這個時機點要徵立馬上工的 Graduate 算是滿少見的情況。

面試共 3 關:

• 第一關 — Online assessment
時間和日期由公司訂好，在測驗開始前 10 分鐘才會把 link 發給所有要參與這次測驗的候選人，測驗的平台是公司自己開發的，測驗期間必須開攝像頭錄影且不能上網查資料，測驗一旦開始就會變成全螢幕模式，如果有切換視窗的動作了話，測驗則會自動送出。

測驗共兩個部分，第一部分是 30 題的多選題，題目涵蓋演算法、資料結構、計算機架構和 SQL，緊接著第二部是 coding test ，Geeksforgeeks 剛好有這題【 Minimum steps to get desired array 】，除了要寫解法外還要自己寫 Unit test，我把解法寫完時間差不多就到了，Unit test 完全來不及寫，送出後就覺得涼了，而且第一部分正確率才 30%，大概沒機會 pass 了。

過幾天 Hunter 就打來恭喜我通過這一輪測驗，我心想「 蛤?這也行? 」，Hunter 表示這階段只有 2 個人 pass，我是其中一個。並跟我約了下一輪 Tech. interview，並跟我說這是 Final stage。

• 第二關 — Tech. interview
面試官是一位 CTO，結果 Tech 的部份是 System design —

「 如果今天要設計一個類似 Instagram 的服務要怎麼做? 」

(幹！Hunter 還騙我說面試內容會討論我第一輪寫的 coding 測驗)，很單純地用口述討論而已，也不需要開白板畫圖。

第二關結束後，Hunter 打來說「 那間公司內部政策有變，所以他們要再多加一輪 Tech. interview。」

• 第三關 — Final tech. interview
面試官還是第二關的那位 CTO，這次面試的內容是 live coding，他先 share 他的螢幕並開了一個 PPT，請我先把題目拍下來，共有三個部分 —

  • 第一部分: SQL
    給兩個 Table，請寫出要找某個資料的 SQL query 語法。

  • 第二部分:Java
    有三小題
    [1] Implement reverse linked list:
    Linked list contains values 101, 90 and 80.

    [2] Check if two arrays contains same elements:
    Integer[] a1 = {1, 2, 3, 2, 1};
    Integer[] a2 = {1, 2, 3};

    [3] White a program to get distinct characters and their count in a String:
    String str1 = “abcdABCDabcd”;

  • 第三部分:
    給一個 Json file，用 Java 讀取 Json file 並將 Json 的內容做排序。

寫完第二部分就差不多剛好一個小時，CTO 便說第三部分我可以 off-line 寫完再寄給他，他已經知道我的 coding 能力了，等他收到我的剩餘的部分會再跟內部討論。結果面試結束沒多久 Hunter 就發 offer 來了，Hunter 便開始追問我有沒有要接受，如果接受了話，叫我 cancel 跟其他家的面試並在 mail 的密件裡 cc 她。

我接受 offer 後便在 mail 裡告訴她

「 因為我先前有被其他公司取消 offer 的經驗，因此在收到正式的合約和 offer 以前我不會取消任何的面試！！ 」

她回我正式的文件幾天後就會下來了，她理解我的擔憂，但她保證不會發生這種事。

她看我不願意取消跟其他家的面試，隔天一早又打電話來跟我吵這件事，一樣又是那句她保證不會發生這種事，我非常不爽的告訴她

「 你的保證對我來說一點屁用都沒有！！！！」

這通電話大概爭執了 20 分鐘，最後我實在被她盧的受不了，便很生氣的說

「 反正我就是不會取消面試啦！！啊不然你現在是想怎樣??要取消我 offer 膩??」

她急忙說道不會因為這樣就取消 offer 啦…

「 所以取消跟其他家的面試這件事到底是來自 “客戶的要求” 還是 “你們 hunter 公司的要求”？」

她這時才終於回答我這個問題，這個我從電話一開始就一直狂問她，但她死不回答我的問題 —

『 這個要求不是直接來自於客戶 』

最後她很無奈的說既然他無論無何都無法說服我，她會盡快請公司把合約寄給我。 這才終於結束這場鬧劇。

你各位看到了沒！！！

這一切都是 Hunter 的話術啦我幹你老師！！！

Hunter 無權要求你做這種事情！！！

要不要取消其他公司的面試完全是你的權利！！！

在你收到正式的 offer 和簽完合約以前，什！麼！都！是！屁！

大概只有菜鳥 hunter 才會做這種事，我跟別家資深的 Hunter 聊起這件事的時候，他們認為不論是候選人在接受 offer 後反悔，在簽了合約後反悔，或是在 onboard 後又跳去其他家，這些都是很稀鬆平常的事，所以根本沒有必要去做這種事情。(他也跟我說，通常候選人要 onboard 滿 3 個月他們才會領到佣金)

英國政府傳送門: 【 Job offers: your rights 】

簽完合約後，應 Hunter 要求，我便寄信並密件 cc Hunter 給一些公司說我要取消面試，不過我都是寄給那些之前面完試就不回的公司來應付 Hunter 。

其實 onboard 後我還是持續面試其他公司 😙
